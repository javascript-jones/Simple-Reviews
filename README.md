# Simple Reviews

Simple Reviews is a WordPress plugin that allows users to leave reviews on businesses. 

## Features

- Adds a "Business" custom post type, with categories and sub-categories, that is searchable
- Allows logged in users to leave a review of these businesses
- Each user can leave one review per business
- Reviews include fields for Review Title, Date of Visit, and the Review text

## Installation

1. Download the repository and unzip the content to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

## Usage

1. After activation, you can add new Businesses from the WordPress admin panel.
2. Users can now leave reviews on Business posts. The review form includes fields for the Review Title, Date of Visit, and the Review text.
